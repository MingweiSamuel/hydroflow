Crate lattices

source ·
Expand description

Module containing lattice traits and implementations.

Convention: Generic parameters that are full words (e.g. Other, Key, Val) are lattice types. Conversely, Generic parameters that are single letters or acronyms (e.g. K, T) are scalar non-Lattice types.
